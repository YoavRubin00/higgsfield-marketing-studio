# Caption Formulas — מבני קופי לפוסט

> נוסחאות קופי שעובדות בעברית לרשתות חברתיות. עיצוב לפי פלטפורמה.
> כל פעם שכותבים פוסט — להתחיל מנוסחה, אחר כך לערוך.

---

## עקרונות כלליים לקופי בעברית

1. **משפטים קצרים**. עברית נוטה למשפטים ארוכים — נלחם בזה.
2. **שורה אחת = רעיון אחד**. רווח בין שורות.
3. **פעלים פעילים**. לא "נעשה X" אלא "תעשה X".
4. **CTA אחד בלבד**. לא 2 לא 3.
5. **לא יותר מדי emoji**. 0-3 בפוסט. לא בכל שורה.
6. **Hashtag-ים בנפרד** (לא בתוך הטקסט). בסוף.

---

## נוסחה 1: PAS — Problem · Agitate · Solution

```
[Problem] שורה אחת — מציינת בעיה ספציפית.

[Agitate] 2-3 שורות — מעמיקה את הבעיה, נוגעת בכאב.

[Solution] 1-2 שורות — הפתרון.

[CTA] שורה אחת — מה לעשות עכשיו.
```

**דוגמה**:
```
מתעורר עייף גם אחרי 8 שעות שינה.

ניסית קפה. ניסית מים קרים. ניסית אפילו ויטמינים.
שום דבר לא עזר.
כי הבעיה היא לא איך אתה מתעורר.

זה לא מה שעושים בבוקר. זה מה שעושים בלילה לפני.

לינק בביו ל-30 ימי ניסיון.
```

## נוסחה 2: Hook · Story · Lesson · CTA

```
[Hook] משפט אחד — תופס תשומת לב.

[Story] 3-5 שורות — סיפור אישי קצר, ספציפי.

[Lesson] 1-2 שורות — מה למדתי / מה התובנה.

[CTA] שאלה / קריאה לפעולה.
```

**דוגמה**:
```
ב-2022 שילמתי יועץ 50,000₪ על משהו שיכולתי לעשות לבד.

באתי למשרד שלו, יושב מולו, מקריא לי טבלת אקסל.
אחרי 3 שעות, חזרתי הביתה, פתחתי את האקסל,
והעתקתי בדיוק את אותם 5 צעדים שעשה.
לקח לי 25 דקות.

אז התחלתי לעשות הכל לבד. לא כי זה זול — כי זה מהיר.

מה הדבר האחרון ששילמת עליו ויכולת לעשות לבד?
```

## נוסחה 3: List Post

```
[Hook] "N דברים ש..."

1. [פריט ראשון]
מקסימום 2-3 שורות הסבר.

2. [פריט שני]
מקסימום 2-3 שורות הסבר.

3. [פריט שלישי]
...

[Outro] שורה — מה הכי חשוב להבין.

[CTA] שאלה / Save.
```

**דוגמה**:
```
3 שגיאות יזמים בשנה הראשונה.

1. רודפים אחרי לקוחות במקום שלקוחות ירדפו אחריהם.
זה ההבדל בין "אני מציע X" ל-"אני המומחה ב-X".

2. מתמחרים את עצמם נמוך כדי לקבל יותר עבודה.
היא לא מקבלת יותר. היא מקבלת לקוחות גרועים.

3. לא מתעדים את התהליך.
חצי מהידע שלך הולך לאיבוד כי לא הוצאת אותו לדפוסים.

מה מהשלוש האלה אתה ב?

שמור את הפוסט אם אתה צריך תזכורת.
```

## נוסחה 4: Counter-Intuitive

```
[Bold Claim] משפט אחד — טענה הפוכה לקונצנזוס.

[Why most are wrong] 2-3 שורות — למה רוב חושבים אחרת.

[Proof] 2-3 שורות — למה הטענה שלך נכונה. ספציפי.

[Implication] 1-2 שורות — מה זה אומר בשבילך.

[CTA] שאלה.
```

**דוגמה**:
```
להעלות מחיר זה הדרך לקבל יותר לקוחות.

רוב היזמים חושבים שמחיר נמוך = יותר לקוחות.
אבל בפועל, מחיר נמוך מסמן "אני זול" — וזה מאיים.

כשהעליתי ב-300%, הלקוחות שלי השתנו: לא יותר חוקרים פיצוצי, אלא לקוחות שיודעים מה הם רוצים.
פחות עבודה, יותר רווח, פחות סבל.

הבעיה אף פעם לא הייתה המחיר. הבעיה הייתה שהפניתי את עצמי לקהל הלא נכון.

אתה מתמחר על "מי הם הלקוחות שלי" או על "כמה שעות עבדתי"?
```

## נוסחה 5: Pattern Interrupt + Story

```
[Surprise] משפט מפתיע / סטטיסטיקה / מצב מוזר.

[Context] 2-3 שורות — מסביר מה קורה.

[Reveal] 2-3 שורות — מגלה את התובנה.

[Personal] 1-2 שורות — איך זה קשור אליך/למשתמש.

[CTA] קצר.
```

## עיצוב לפי פלטפורמה

### Instagram (פוסט מתחת לתמונה)
- **אורך**: 100-300 מילים אופטימלי
- **שורה ראשונה**: חזקה — "...עוד" יחתוך אחרי ~125 תווים
- **רווחי שורה**: כן, חיוני
- **Hashtags**: 5-15, בסוף, בקבוצה. עברית: #יזמות #שיווק וכו'
- **Emoji**: 0-2

### TikTok (caption)
- **אורך**: 50-150 תווים אידיאלי
- **Hook ב-caption**: כן — מסקרן את מי שלא לחץ אודיו
- **Hashtags**: 3-5, מעורבב עברית/אנגלית
- **Emoji**: 0-1

### Facebook
- **אורך**: 50-200 מילים
- **שורה ראשונה הכי חשובה** — אחר כך "ראה עוד"
- **Hashtags**: 0-3 (Facebook לא מאוד hashtag-friendly)
- **לינקים בקומנט הראשון** (אלגוריתם פוסט עם לינק = downgrade)

### Twitter/X
- **אורך**: 280 תווים — או thread
- **Thread**: כל ציוץ 1-2 משפטים, ממוספר אם יש סדר
- **Hashtags**: 0-2, רק אם רלוונטי
- **CTA**: ב-tweet האחרון של thread

### LinkedIn
- **אורך**: 150-400 מילים
- **שורה ראשונה**: hook — אחר כך "...see more"
- **Hashtags**: 3-5, מקצועיים
- **Emoji**: זהיר, רק אם הקהל קליל
- **כתיבה רשמית יותר** מאינסטגרם

## CTAs מומלצים בעברית

### לעירוב (engagement)
- "ספר לי בקומנט: ..."
- "מסכים? לא מסכים?"
- "אם זה דיבר אליך — תגיד 'כן' למטה."
- "תייג מישהו שצריך לראות את זה."

### לשמירה (saves)
- "שמור את הפוסט — תרצה אותו אחר כך."
- "הוסף לפיברטיס — עוד תחזור לזה."

### למעבר ללינק
- "לינק בביו."
- "ראה עוד בלינק."
- "פרטים בלינק שבסטוריז."

### לרכישה (זהיר — לא בכל פוסט!)
- "זמין עכשיו ב__"
- "להזמנה בלינק בביו."

## הערה על AI-jargon

הימנע ממילים שמסגירות שזה כתוב ב-AI:
- "מהפכני" / "פורץ דרך"
- "פתרון הוליסטי" / "סינרגיה"
- "חוויה ייחודית" / "מותאמת אישית"
- "delve into" / "leverage" (גם בעברית מתורגם)

הריץ `de-ai-ify` לפני פרסום אם לא בטוח.
