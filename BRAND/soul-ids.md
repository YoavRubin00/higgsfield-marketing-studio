## דמויות פעילות

> **אין Soul ID פעיל כרגע.** ראה המלצה אסטרטגית למטה לפני שמתחילים.

### המלצה אסטרטגית — מתי לאמן Soul ID

> **מקור**: [../ANALYTICS/instagram-baseline-2026-05.md](../ANALYTICS/instagram-baseline-2026-05.md)

הריל הזוכה הגדול של החשבון (181 לייקים, 67 תגובות — "המענק נכנס") השתמש ב**פרזנטור אנושי אמיתי בשטח חיצוני** — לא Soul ID, לא קפטן שארק, לא אווטאר Higgsfield. תגובות מסוג "מי החתיך הזה?" מוכיחות שהאותנטיות של אדם אמיתי הניעה את הביצועים.

**מסקנה**: לא ליצור Soul ID לפני שיש לפחות **3 ריילים מאומתים** (50+ לייקים כל אחד) עם הפרזנטור הקבוע. אחרי שיש דפוס מוכח, אז כדאי לאמן Soul ID כדי לאפשר scaling (פרזנטור באירועים שלא הולכים לצלם בשטח).

### מסלול מומלץ לכניסה ל-Soul ID

1. **שלב 1 (עכשיו)**: 3-5 ריילים אמיתיים עם הפרזנטור (כל ריל אקטואלי בעמוד 1)
2. **שלב 2 (אחרי הוכחת דפוס)**: לאמן Soul של אותו פרזנטור (10-20 תמונות איכותיות)
3. **שלב 3**: להריץ generation דמו על אותו hook שעבד בשטח — להשוות איכות
4. **שלב 4**: רק אם ה-Soul ID מצליח להחזיק 70%+ מהביצועים האנושיים — לעבור ל-scaling
5. **תיעוד**: למלא כאן את ה-`reference_id` ולעקוב אחרי דמו generation

**אזהרה**: לא להחליף פרזנטור אמיתי ב-Soul ID לפני שיש הוכחה. הפרזנטור האנושי ברחוב הוא ה-edge הגדול של החשבון.

### Soul ID לקפטן שארק?

**לא נדרש כרגע** — קפטן שארק נוצר מצוין דרך פרומפטים ב-Nano Banana / GPT Image (ראה [prompt-library.md](prompt-library.md#captain-shark--character-prompts-nano-banana--gpt-image)). אם בעתיד יוצג חוסר עקביות בין פוסטים, לשקול לאמן Soul של דמות הקפטן.

## הוראות לאימון Soul חדש

1. **לאסוף 10-20 תמונות איכותיות** של הדמות:
   - זוויות שונות (פרונטלי, פרופיל, 3/4)
   - תאורה משתנה (יום/לילה, טבעית/סטודיו)
   - הבעות שונות (חיוך, רצינות, ניטרלי)
   - **כולל full-body** אחת לפחות (לפרופורציות)
   - **בלי משקפי שמש** או צללים כבדים על הפנים
   - תמונות מ-4-5 חודשים אחרונים (העדפה)

2. **לשמור ב-`ASSETS/voices/<character-name>/`** (כן, זו תיקיית `voices/` משמשת גם להצליל וגם לתמונות מקור Soul — נשנה אם נצטרך)

3. **להריץ סקיל**: `higgsfield-soul-id` עם הנתיב לתיקייה.

4. **לתעד כאן** את ה-`reference_id` שחזר.

5. **לבדוק**: להריץ generation דמו ולוודא עקביות.

## עלות ידועה

- אימון Soul חדש: ~$3 (כ-200 קרדיטים) — לא ממש קריטי, אבל לזכור.
- generation עם soul-id: עלות רגילה לפי המודל שמשתמשים.
