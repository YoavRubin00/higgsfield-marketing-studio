## ElevenLabs — קריינות עברית

### הגדרות שעבדו
- **קול**: <להוסיף שם voice>
- **stability**: 0.5 (ניטרלי), 0.7 (יותר עקבי)
- **clarity**: 0.75
- **style**: 0.3 (טון רגיל)

### Tips
- לכתוב מספרים במילים: "שלוש" במקום "3"
- לסמן הפסקות עם פסיקים ופוינטים
- לבדוק תוצאה עם דגימה קצרה לפני שיוצרים את הקול המלא

## קופי — formulas שעובדים

### Hook patterns
1. "איש לא מספר לך ש..."
2. "אם הייתי יודע את זה לפני שנה..."
3. "שאלו אותי איך X — והתשובה הפתיעה אותי"

(להרחיב לפי מה שעובד)

### Body structures
- Problem → Agitate → Solution → CTA
- Hook → Story → Lesson → CTA
- Question → Answer → Reframe → CTA
